## ğŸ® How to Play

1. **Start the Game**
   - Click "Let's Play!" on the landing page
   - Choose your preferences (Dark Mode, Music)
   - Select a difficulty level (Easy, Medium, or Hard)
   - Choose a theme (Fruits, Animals, or Shapes)
   - Click "Start Game âœ¨"

2. **Gameplay**
   - Click on cards to flip them and reveal the emoji
   - Find matching pairs of emojis
   - Match all pairs before time runs out
   - Use the "Reveal All" button once per game for help

3. **Controls**
   - **Pause Button**: Pause and resume the game
   - **Reveal All**: See all cards for 3 seconds (one-time use)
   - **Back to Menu**: Return to game settings
   - **ESC Key**: Quick exit to menu (keyboard shortcut)

4. **Winning**
   - Match all pairs before the timer reaches zero
   - Enjoy the spectacular victory animation!
   - Try to beat your score with fewer moves
